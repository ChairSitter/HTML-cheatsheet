Introduction to HTML tags

HTML is a programming language used to organize and format the content of a website.

HTML uses "tags" inside of angle brackets to indicate elements that organize and separate a document into sections.

10 common HTML elements and their tags:

Element Name Tag Description
html <html> Included in every html document as the root element that defines the html document.
head <head> Descendant of the html element. It is used for the title and for metadata that the end user won't see.
body <body> Descends from the html element, along with the head tag. It contains all the visible contents of the website.
header <heading> Descends from the body element, along with the main and footer elements. Contains heading content for the website.
main <main> Descends from the body element, along with the header and footer elements. Contains the main or central content of the website.
footer <footer> Descends from the body element, along with the header and main elements. Contains the footer or end content of the website.
title <title> Descendant of the head element. Contains the title of the website that will appear on the browser tab or window.
h1-h6 <h1>-<h6> Heading elements ranging from h1 to h6, with lower numbers being more important and generating larger font sizes.
paragraph <p> Paragraph element allowing general text to be added to the site.
division <div> Versatile element used to divide and organize sections of information. Should not be overused.
